# Shelby Ecosystem Analytics Dashboard

An experimental analytics and discovery dashboard for the Shelby ecosystem on Aptos.

## Overview
This project aims to help developers and users discover new tools, builders and ecosystem activity happening across Shelby.

## Features (Planned)
- Track new builders joining the ecosystem
- Highlight early stage projects
- Monitor ecosystem activity
- Provide ecosystem growth insights

## Status
Currently exploring ideas and preparing the initial project structure.
